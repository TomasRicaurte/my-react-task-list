#¿Qué es mi producto y para que sirve?

Mi producto es una lista de tareas interactiva con los usuarios que la manejen,
con la posibilidad de poder crear nuevas tareas con un titulo y descripción que ayudaran a comprender
mejor el que hacer para resolver cada una.

#¿Cuáles son las funcionalidades más importantes y porque los usuarios las usarían?

Las funcionalidades más importantes que posee la lista de tareas proporcionada por el producto
son las siguientes:

> Creación de tareas con titulo y descripción unicas para su mayor desglose.
> Edición de las tareas en su totalidad para poder modificarlas para su conveniencia.
> Eliminación de tareas que ya hayan sido completadas o ya no se vayan a realizar para mantener un mejor orden.
